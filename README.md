# Inventory Manager

## Project Overview

Inventory Manager is a web-based Inventory Management System designed to simplify stock handling, category organization, and product tracking. The application provides a user-friendly interface with backend integration for managing inventory efficiently.

## Key Features

- User Authentication and secure login
- Dashboard for quick inventory overview
- Product Management (Add / Update / Delete / View)
- Category Management
- Inventory Tracking
- Responsive frontend pages
- Spring Boot backend APIs
- Structured and scalable codebase

## Installation & Setup

```bash
git clone https://github.com/Jiyaa22/InventoryManager.git
cd InventoryManager
mvn spring-boot:run
```

Open in browser:

```text
http://localhost:8081/index.html
```
